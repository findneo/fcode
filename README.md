# fcode
概述：
  这个小工具灵感来自于一个[在线编解码工具](http://tool.ph0en1x.com/hashtool/tools.html#conv/) ，是学习Tkinter时的副产品。目前还比较粗糙，不过已经基本可用。

功能：
实时互转ASCII，BASE64，BASE32，HEX(BASE16)，DEC，OCT，INT，URL ENCODE，HTML ENCODE，MORSE，ROT，栅栏密码等

快捷键：
* ESC 退出
* TAB 切换
* SHIFT+TAB 切换
* ENTER 全选
* CTRL+X/C/V  剪切/复制/粘贴

依赖：
Python2
  * Tkinter
  * base64
  * re
  * hashlib
